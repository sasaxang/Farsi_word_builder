## How to Capture
1. Open your browser to http://localhost:8501
2. Use browser's screenshot tool or snipping tool
3. Save screenshots with the exact names listed above
4. Save all screenshots in: `/home/sasaxang/Farsi_word_builder/docs/screenshots/`

## Tips
- Capture at a consistent window size for uniformity
- Make sure all text is clearly readable
- For before/after comparisons, use the same word
- Highlight or annotate key features if needed
